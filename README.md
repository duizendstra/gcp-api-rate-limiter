# gcp-api-queue
Google Cloud Platform API queue
=========

No dependencies rate limiter for the Google Cloud Platform 

## Installation

  `npm install gsp-api-rate-limiter`

## Usage

    let rateLimiter = gcpApiRateLimiter({
         tasksPerMillisecond: 1000
    });
